### Usage:
```lua
local yesplib = loadstring(game:HttpGet("https://raw.githubusercontent.com/haes0wbgknqoy1004r4n/roblox-esplib/refs/heads/main/lib.lua"))()
yesplib:Init()
```

### Configuration:
## Example:
```lua
yesplib.settings.friendly.boxEnabled = true
```
